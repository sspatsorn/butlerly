# AI Task Butler

สร้าง AI เลขาส่วนตัวบน LINE ที่ช่วยจัดการงานให้โดยอัตโนมัติ

## เป้าหมาย

ผู้ใช้เพียงส่งหรือ Forward ข้อความงานมาหา Bot แล้ว AI จัดการทุกอย่างให้ ตั้งแต่สร้าง Task, วิเคราะห์ Deadline, สร้าง Checklist, บันทึกข้อมูล และแจ้งเตือนผ่าน LINE

## ฟีเจอร์หลัก

* รับข้อความหรือ Forward งานจาก LINE
* ใช้ Google Gemini วิเคราะห์ข้อความ
* สร้าง Task อัตโนมัติ
* แยก Deadline และเวลา
* สร้าง Checklist จากรายละเอียดงาน
* บันทึกข้อมูลลง Supabase PostgreSQL
* แจ้งเตือนผ่าน LINE ตามเวลาที่กำหนด
* รองรับการสนทนาด้วยภาษาธรรมชาติ เช่น

  * วันนี้มีงานอะไร
  * งานนี้เสร็จแล้ว
  * เลื่อนไปพรุ่งนี้
  * เตือนอีก 30 นาที
* รองรับหลายผู้ใช้ โดยใช้ LINE User ID เป็นตัวแยกข้อมูล

## Tech Stack

* Nuxt 3
* Express
* Supabase PostgreSQL
* Google Gemini API
* LINE Messaging API

## สิ่งที่ต้องการให้พัฒนา

พัฒนาโปรเจกต์แบบ Full Stack พร้อมโครงสร้างที่ขยายต่อได้ แยก Frontend, Backend และ Database อย่างเป็นระบบ ใช้โค้ดที่สะอาด (Clean Architecture) และสามารถรันได้จริง

เป้าหมายของระบบคือ

**"ส่งงานมาให้ AI ที่เหลือ AI จัดการให้ทั้งหมด"**
